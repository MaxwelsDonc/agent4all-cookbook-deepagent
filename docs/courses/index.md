# 课程总览

Agent4all 会分成 5 门课程。

整套课程只围绕一个贯穿项目：

> MaxLance Assistant，一个帮助初学者学习 DeepAgent 的学习助手。

你会从“看懂它是什么”开始，最后做出一个本地可运行、可继续改造的学习助手。

## 开始前先知道

<div class="a4a-two-column">
  <div>
    <h3>这条路线适合你，如果你想要</h3>
    <p>从零理解 Agent 和 Deep Agents，跟着教程跑通代码，并一步一步做出自己的 Agent 项目。</p>
  </div>
  <div>
    <h3>这条路线暂时不会做</h3>
    <p>一开始就讲生产级部署、解释所有 LangChain 内部机制，或者让 Agent 随便读写你的电脑。</p>
  </div>
</div>

## 每一节怎么学

你不需要背模板。

只要记住读每一节时都问自己四个问题：

1. 这是什么？
2. 为什么要学？
3. 我现在怎么做？
4. 我怎么知道自己做对了？

如果一节课没有让你回答出这四个问题，就说明这一节还需要继续改。

## 从哪里开始

第一次学习建议从 [01. DeepAgent 零基础入门](./01-basics/) 开始。

如果你已经理解 Agent 和 Deep Agents 的大概关系，可以从 [02. 从 0 跑通第一个 Agent](./02-first-agent/) 开始。
