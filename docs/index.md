# Agent4all

让 DeepAgent 不再 deep。

这是一条面向初学者的学习路径，围绕一个贯穿项目 —— **MaxLance Assistant**，从概念、环境、工具到安全项目，一步步做出一个能规划、用工具、保存结果并保持安全边界的本地学习助手。

你不会每章重做一个 demo，而是持续改造同一个助手。

## 怎么读这套教程

每一节都围绕四个问题展开（标题不会机械地这么写）：

> 这是什么？ · 为什么要学？ · 现在怎么做？ · 怎么知道做对了？

如果一节没让你回答出这四个问题，就说明它还需要继续改。

## 从哪里开始

第一次学习，建议从 [课程总览](/courses/) 开始；它列出五门课的顺序，以及每一门学完后你会得到什么。

如果你已经理解 Agent 和 Deep Agents 的关系，可以直接进入 [从 0 跑通第一个 Agent](/courses/02-first-agent/)。
